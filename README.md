# PowerCycle-Tester

전원 On/Off 테스트 보드 펌웨어 및 PC 제어 프로그램

## 개요

PowerCycle-Tester는 제품의 전원 사이클 테스트를 자동화하는 테스트 보드용 펌웨어와 PC 제어 프로그램입니다.
